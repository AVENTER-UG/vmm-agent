# Agent for MicroVM's

This is a fork of: https://github.com/codebench-dev/agent


## Build

### Build Linux Kernel

```bash
make kernel
```

### Build Rootfs

```bash
make rootfs
```

### Start MicroVM

```bash
make start-vm
```

### Stop MicroVM

```bash
make stop-vm
```
